# Homebrew Tap for vcon-dev

This tap contains Homebrew formulas for vcon-dev tools.

## Installation

```bash
brew tap vcon-dev/tap
brew install conserver
```

## Available Formulas

### conserver

CLI tool to manage vcon-server Docker containers.

```bash
brew install vcon-dev/tap/conserver
```

## Updating

To update formulas:

```bash
brew update
brew upgrade conserver
```
